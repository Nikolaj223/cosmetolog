# Beauty Lab Demo

Учебный сайт косметологической клиники «Лаборатория красоты», собранный на `HTML`, `CSS` и `vanilla JS`.

## Что внутри

- Публичный сайт с блоками услуг, специалистов, портфолио, отзывов, акций и контактов.
- Многошаговая онлайн-запись с выбором услуги, специалиста, даты и времени.
- Админ-панель с локальной авторизацией, редактированием контента, модерацией отзывов, управлением расписанием и экспортом записей в CSV.
- Локальное хранилище на базе `localStorage`, чтобы без сервера показать сценарий, близкий к БД.

## Демо-доступ в админку

- Логин: `admin`
- Пароль: `beauty-lab2026`

## Как запустить

1. Можно открыть [index.html](c:/da/index.html) напрямую в браузере.
2. Если браузер ограничивает `localStorage` для `file://`, запустите локальный сервер:

```powershell
powershell -ExecutionPolicy Bypass -File .\serve.ps1
```

После этого откройте `http://localhost:8080`.

## Важное допущение

По ТЗ предполагается `SQLite` и серверные уведомления, но в текущем окружении нет `Node.js`. Поэтому для рабочей демо-версии запись клиентов и управление контентом реализованы на чистом фронтенде через `localStorage`. Структура приложения сделана так, чтобы позже можно было заменить слой хранения на API и реальную БД.
